#### **USER Story:**

**As a user,**
**I want to** customize the header columns of the Insurance listing page
**So that** I can show, hide, and reorder relevant columns according to my preferences.

---

#### **acceptance criteria:**

1. A **Show/Hide** icon must be available in the top-right corner of the Insurance listing page column titles row.

2. On clicking, two tabs must be displayed:
   * **Show/Hide**
   * **Reorder**

3. The **Show/Hide** tab must include:
   * Search field
   * Update button
   * Select All option
   * List of column titles with checkboxes
   * The user can check or uncheck any column title to control its visibility on the Insurance listing page.

4. The **Reorder** tab must include:
   * Reset button
   * Update button
   * List of column titles
   * The user can rearrange column titles by dragging them upward or downward.
   * Once updated, the new order must be reflected on the Insurance listing page.

5. The following column titles must be included in both Show/Hide and Reorder:

   * Category
   * Linked Entity
   * Type of Cover
   * Insurance Type / Clause Type
   * Policy Number
   * Policy / Cover Reference
   * Currency
   * Deductible
   * Sum Insured
   * Total Sum Insured (TSI)
   * Annual Premium
   * Broker
   * Broker Reference Number
   * Broker Contact
   * Insurer / Club
   * Leading Underwriter
   * Insurer Contact
   * Date of Notification to Broker
   * Policy Start Date
   * Policy End Date
   * Insurance Status
   * Created By
   * Remarks
   * Created Date

6. **ID** and **Insurance No** are mandatory columns and must not be available in the Show/Hide or Reorder options — they are always visible and always fixed in their position.

7. Both Show/Hide and Reorder functionalities must preserve existing behaviour and user experience.
