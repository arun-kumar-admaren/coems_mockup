#### **USER Story:**

**As a user,**
**I want to** customize the header columns of the Legal Review listing page
**So that** I can show, hide, and reorder relevant columns according to my preferences.

#### **acceptance criteria:**

* A **Show/Hide** icon must be available in the top-right corner of the Legal Review listing page.
* On clicking, two tabs should be displayed:
  * Show/Hide
  * Reorder
* The **Show/Hide** tab must include:
  * Search field
  * Update button
  * Select All option
  * List of column titles with checkboxes
* The user can check/uncheck any title to control visibility on the Legal Review listing page.
* The **Reorder** tab must include:
  * Reset button
  * Update button
  * List of column titles
* The user can rearrange column titles by dragging them upward or downward.
* Once updated, the new order must be reflected on the Legal Review listing page.
* The following column titles must be included:
  * Related To
  * Items
  * Review Type
  * Description
  * Legal Review Status
  * E Filing Number
  * Due Date
  * Review Raised By
  * To Be Reviewed By
  * Created Date
* **ID** and **Review No** are mandatory columns and will not be available in the Show/Hide or Reorder options.
* Both Show/Hide and Reorder functionalities must preserve existing behaviour and user experience.
